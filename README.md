# columns.asm
Implementation of the original columns game using MIPS assembly.


# How to play
- Download Mars / Saturn IDE
- Open columns.asm in the IDE and run!

## Controls:
- **1, 2 or 3**: select difficulty
- **A/D**: move column left/right
- **W**: shuffle gems
- **S**: move column downwards
- **P**: pause game
- **R**: retry
- **Q**: quit
